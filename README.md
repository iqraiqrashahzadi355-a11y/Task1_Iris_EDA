# Task 1: Iris Dataset EDA

## Objective
Explore and visualize the Iris dataset to understand data trends, distributions, and feature relationships.

## Dataset
- Iris Dataset (built-in Seaborn dataset)
- 150 samples, 5 columns: `sepal_length`, `sepal_width`, `petal_length`, `petal_width`, `species`

## Steps
1. Load dataset using pandas
2. Explore data with `.head()`, `.info()`, `.describe()`
3. Visualize data using:
   - Scatter plot (Sepal Length vs Sepal Width by Species)
   - Histogram (Petal Length distribution)
   - Box plot (Sepal Length by Species)

## Key Insights
- Scatter plot clearly shows separation of species clusters
- Histogram shows distribution of petal lengths
- Box plot identifies outliers and species-wise comparison

## Libraries Used
- pandas
- numpy
- matplotlib
- seaborn
